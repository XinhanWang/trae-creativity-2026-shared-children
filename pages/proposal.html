01 · 创意概述

用"共享儿女"的方式，重新定义老年陪伴

当 60 岁以上人口突破 2.8 亿，独居与空巢老人数量持续攀升，"谁来陪爸妈说话、跑跑腿"正在变成一个社会化问题。

要解决什么问题

中国有近 1.18 亿 独居和空巢老人[1]。他们中很多人不缺物质，缺的是"被陪伴"：有人聊聊天、有人帮取药、有人教用智能手机。子女在外地、护工费用高昂且资质参差，传统的"家政 + 社区服务"模式又难以解决情感层面的孤独。

为什么会想到做这个

美国的 Papa（Y Combinator 2018 夏季批次，业内简称 YCS18）开创了"Family on Demand"模式：把经过筛选的"陪友"（Papa Pals）按需匹配给老年人家庭，提供陪伴、跑腿、就医接送等服务，估值一度达到独角兽级别[2]。这启发我们：中国有更密集的青年志愿者群体（大学生、退休低龄老人、社区热心人），如果能在中国本地化这套"平台 + 严选 + 灵活性匹配"的机制，就能把原本昂贵的"私人陪护"变成普惠型的社会服务。

大概是什么产品

一款基于微信小程序的"共享儿女"平台，连接 "暖伴"（青年志愿者） 与 "暖家"（老人家庭），提供三类核心服务：日常陪伴、跑腿代办、智能手机与就医协助。配套 Web 后台用于志愿者筛选、订单调度、安全审计与数据统计。

02 · 目标用户与痛点

两类用户，三方共赢

我们不是要做"替老管家"，而是要用平台化方式，让青年、老人、社区都从中受益。

面向哪些用户

家

暖家（银发家庭）

60+ 独居或空巢长者，子女异地、行动不便、对数字工具陌生。需求是有人说话、有人帮忙、有人兜底。

伴

暖伴（青年志愿者）

大学生、社区青年、低龄退休人员、心理辅导师、社工等专业群体。动机包括公益时长、社会实践、收入补贴。

社

暖社区（街道/物业/医院）

基层政府、街道办、社工站、医院导诊科、企业工会。可以用平台把分散的志愿资源统一调度。

在什么场景下使用

  • 01 工作日中午，子女在外上班，独居父亲不会网上挂号，志愿者代挂号并陪同就诊。
  • 02 周末午后，青年志愿者上门陪老人下棋、读报、拍一段全家福短视频远程发给孩子。
  • 03 突发情况，老人血压飙高需要跑腿买药，平台 30 分钟内匹配最近志愿者上门。
  • 04 临年过节，子女因病情或工作无法回家，志愿者上门贴春联、陪吃年夜饭。
  • 05 智能手机不会用，教老人发视频、打车、缴水电费，缓解数字鸿沟焦虑。

当前痛点（没有我们的产品时）

对老人来说

  • 请专业护工：每月 4000-8000 元，且质量参差、情感连接弱。
  • 靠社区上门：频次低、覆盖窄、节假日基本停摆。
  • 靠子女远程视频：情感温度有限，无法解决"跑腿"刚需。
  • 自己硬扛：安全隐患大，突发疾病难以及时发现。

对青年志愿者来说

  • 想做公益找不到可信的入口：怕被骗、怕担责、怕被骚扰。
  • 零散志愿活动难以持续：没有稳定的"工单池"和回报机制。
  • 公益时长认证繁琐：高校、企业、街道各一套系统。
  • 缺少培训与意外保障：想做但不敢做。

03 · 价值与意义

不止是生意，更是基础设施

从社会价值、商业价值、效率提升三个角度，我们看到这件事有成为"银发时代基础设施"的潜力。

社会价值

把"陪伴"从失灵商品变成普惠品。一个青年志愿者每周 4 小时，一年可服务近 200 小时，覆盖 3-5 个家庭，直接缓解城市独居老人的情感孤独与安全盲区。参考 Papa 的数据，其用户家庭的就医依从性、慢病管理依从性显著提升，间接降低医保支出[2]。

商业价值

中国银发经济规模已超 7 万亿元[3]，但"情感 + 跑腿"赛道仍处于早期。我们采用 B2B2C 模式：企业工会（员工父母关爱福利）、商业保险公司（增值服务）、街道社区（政府采购）作为付费方，老年家庭作为受益方；志愿者按单获得补贴 + 公益时长，形成多方可持续闭环。

效率提升

相比传统 1 对 1 上门家政，平台化匹配可让志愿者的"在路上 + 上门"时间从 60% 提升到 85% 以上，单志愿者一年可服务的家庭数提升 3-4 倍，间接压低单位服务成本。

05 · 解决方案

一套"平台 + 严选 + 灵活"的中国版 Papa

我们把 Papa 验证过的"Family on Demand"模式，翻译成更适合中国本地的形态。

01

暖家下单

子女或老人在小程序发起订单，类型、时长、地点、人数一目了然。

02

智能匹配

系统按距离、技能、评分、性格标签 3 分钟内推荐"暖伴"。

03

上门服务

志愿者按约定时间上门，全程录音/定位，保障双方安全。

04

双向评价

服务结束后双向打分，形成暖伴信用档案，沉淀家庭健康档案。

核心功能模块

日常陪伴

聊天、读报、陪下棋、节日陪伴、心理疏导。最少 1 小时起，可按周/月预约。

跑腿代办

代取药、代买菜、代缴水电费、陪同就医、物件代取。按"次/公斤/件"计价。

数字辅导

教用智能手机、视频通话、防诈窗提醒、健康码操作。30 分钟一课，可远程。

安全兜底

志愿者实名 + 背调 + 保险 + 一键报警 + 服务全程录音定位。

家庭健康档案

脱敏记录服务频次、用药提醒、情绪曲线，反馈给子女与社区医生。

B 端采购

企业工会、保险公司、街道办可批量采购，作为员工父母关爱或公共服务。

06 · 典型场景

两个具体的陪伴时刻

产品最终要落到真实生活里。以下两个场景，来源于对武汉市独居家庭的访谈。

场景 A · 周末公园的代闲散步

李奶奶，72 岁，独居，腿脚不便。儿子在深圳工作，每两周视频一次。

  • 儿子在平台预约：每周六上午 9:00-11:00，公园陪伴 2 小时。
  • 平台匹配附近的大学生"暖伴"小张，固定服务 8 周。
  • 服务结束后生成一段短视频，自动同步给儿子作为"陪伴证据"。
  • 儿子月度账单 200 元，可由企业工会全额报销。

场景 B · 早高峰的代取药

王大爷，68 岁，糖尿病 + 高血压，需长期服药。女儿在外地无法陪同。

  • 女儿在小程序发起"代取药 + 陪同挂号"，定位离家最近的三甲医院。
  • 平台 12 分钟内匹配"暖伴"李姨（退休护士），自带专业背包。
  • 李姨完成挂号、缴费、取药、复诊建议全程，生成电子档案。
  • 若李姨出现不适，一键呼叫家庭医生与社区应急联动。
07 · 商业模式

三方付费，可持续飞轮

我们坚持 B 端付费为主、C 端补贴为辅，让真正需要陪伴的老人家庭几乎免费享受服务。

付费方 使用场景 典型客单价 价值
企业工会 员工父母关爱福利，团队增值 300-500 元/家庭/月 雇主品牌 + ESG
商业保险 长护险增值、慢病管理 按保单规模分摊 降低赔付率
街道/民政 政府购买公共服务 50-100 元/次 补足基层服务短板
个人 异地子女为父母购买 100-200 元/次 情感刚需 + 应急兜底

志愿者回报设计

  • ✓ 基础补贴：每单 30-80 元（视类型），覆盖路费与时间成本。
  • ✓ 公益时长：自动同步至全国志愿服务信息系统，高校可认证。
  • ✓ 能力成长：免费培训、心理学习、适老化沟通培训，结业发证。
  • ✓ 职业通道：累计服务满 500 小时可申请成为"金牌暖伴"，单价上浮 30%。
08 · 本地化与护城河

不是另一个Papa 复刻

我们看到 Papa 模式在中国落地有 3 个关键变量，必须做本地化重构。

A

更密的志愿者池

中国有 1.7 亿在校大学生 + 1.5 亿低龄退休人员，志愿者供给密度远高于美国。

B

更强的政企协同

街道/工会/保险天然具有集中采购能力，B 端上量速度比 C 端裂变更快。

C

更轻的产品形态

基于微信小程序而非独立 App，零安装成本，对老人极其友好。

核心护城河

  • ✓ 信任护城河：实名 + 公安背调 + 双向评价 + 保险，构建平台信用护城河。
  • ✓ 数据护城河：沉淀中国首个"城市级银发陪伴行为数据库"，反向精准匹配与健康预警。
  • ✓ 渠道护城河：与街道办、工会、保险公司深度合作，新城市启动周期可压缩至 4 周。
